# 🏕️ START HERE

Welcome to the camping trip itinerary website! This file will get you started in under 5 minutes.

## Quick Start (Copy & Paste These Commands)

### 1. Install dependencies
```bash
npm install
```

### 2. Start the development server
```bash
npm run dev
```

### 3. Open your browser
```
http://localhost:3000
```

That's it! You should see the beautiful itinerary website.

---

## 📝 Making Your First Edit

1. Open `src/data/itinerary.ts` in your text editor
2. Find an activity you want to change
3. Change the text
4. Save the file (Ctrl+S or Cmd+S)
5. The website auto-updates in your browser!

### Example: Change an activity time

Find this:
```typescript
{
  time: '11:30 AM',
  title: 'Arrive in Paso Robles',
```

Change to:
```typescript
{
  time: '12:00 PM',
  title: 'Arrive in Paso Robles - Coffee Break',
```

Save and see the change instantly!

## 🎨 Customize the Colors

Edit `tailwind.config.ts` to change the color palette:

```typescript

  50: '#faf8f3',  // Change these hex colors
  // ...
}
```

Or edit day-specific colors in `src/app/page.tsx`.
